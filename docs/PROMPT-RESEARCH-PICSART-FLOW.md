# Nghiên cứu Picsart Flow

## Yêu cầu

Dùng agent skill nghiên cứu (web search, đọc docs, truy cập trang web) để tìm hiểu chi tiết về **Picsart Flow** (https://picsart.com/flow/).

Đây là nền tảng AI workflow no-code dạng node-based canvas — cho phép kéo thả các khối xử lý (node) thành pipeline sáng tạo.

## Nội dung cần nghiên cứu

### 1. Tổng quan Picsart Flow
- Nó hoạt động thế nào? Canvas, node, connection giữa các node
- Các loại node có sẵn (text, image gen, video gen, remove bg, upscale, voiceover, sound...)
- AI model nào được tích hợp sẵn? (Kling, Veo, Runway, Flux, SDXL...)
- Free tier cho phép gì, paid tier giá bao nhiêu, AI credits hoạt động ra sao

### 2. Workflow cho video affiliate TikTok Shop
Bối cảnh: User làm affiliate marketing TikTok Shop VN. Có ảnh sản phẩm gốc + thông tin SP. Cần tạo video bán hàng ngắn (15-60s) để đăng TikTok.

Nghiên cứu:
- Có thể tạo workflow từ **ảnh SP → video bán hàng** hoàn chỉnh trên Picsart Flow không?
- Các node nào cần dùng, thứ tự ra sao
- Output video chất lượng thế nào
- Có template workflow sẵn cho e-commerce / product video không

### 3. Các kiểu video có thể tạo
- Slideshow ảnh SP + nhạc + text overlay
- AI-generated video từ ảnh SP (image-to-video)
- Before/after, demo, lifestyle
- Voice-over + ảnh/video SP
- Kiểu nào Picsart Flow làm tốt, kiểu nào không

### 4. API / Automation
- Picsart Flow có API không? Có thể trigger workflow từ bên ngoài không?
- Có thể share workflow / template không?
- Có batch processing không (chạy 1 workflow cho nhiều SP)?

### 5. So sánh nhanh
- So với Freepik Spaces và Weavy, Picsart Flow mạnh/yếu ở điểm nào cho use case affiliate video

## Output

Tạo file `RESEARCH-PICSART-FLOW.md` với:
- Mô tả chi tiết cách Picsart Flow hoạt động
- Danh sách các node types có sẵn
- 2-3 workflow mẫu cho video affiliate (step by step, node nào → node nào)
- Pricing và giới hạn
- Đánh giá: có phù hợp để tích hợp vào PASTR không, nếu có thì tích hợp ở điểm nào

Viết báo cáo bằng tiếng Việt.
